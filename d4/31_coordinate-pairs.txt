> Query: Input contains the x coordinate followed by the y coordinate of a
> point.  Output the pair (x, y) together.
> Example:
> [input: 2, 5, 0, 4]
> [output: (2,5), (0,4)]
// Read the x-coordinate, followed by the y-coordinate.  Print
// the coordinate pair.  This solution is optimal, according
// to the game.
int = input;        // Read the x-coordinate.
reg = input;        // Read the y-coordinate.
output = int, reg;  // Print the point (x, y).
